# Screenshot Assets

Expected files:

- `dashboard.png`
- `series-detail.png`
- `compare-page.png`
- `mobile-view.png`

To capture them locally:

1. Run `docker compose up --build -d`.
2. Open http://localhost:5173.
3. Capture desktop screenshots for dashboard, series detail and compare pages.
4. Use browser device mode for a mobile screenshot.
5. Save the files into this folder with the names above.

The README already reserves these slots so the project can be published before final visual assets are captured.
